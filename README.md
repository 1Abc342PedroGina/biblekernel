# biblekernel
The BibleOS and derivates Core

It's a hybrid file system; the core is a microkernel, the security is provided by an exokernel combined with a nanokernel. The monolithic system works with kernel driver components and communication between POSIX processes (i.e., Windows NT KTHREADS, EPROCESS, ETHREADS, and KPROCESS, QNX Threads, Sys-v/BSD Process (Procs and Threads), Linux Tasks (Proc, Cgroup, Tasks), Mach Process (Task, Thread), DOS Procs, Windows 9x VMM Kernel Threads and Process, Solaris LWPs, SEL4 VSpace + CSpaces + TCB + Scheduling Contexts (MCS)) and the file system.
